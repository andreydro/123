Что такое контекст вызова функции? Чем определяется?    --- Код в JavaScript может быть одного из следующих типов:
																eval-код — код, выполняющийся внутри функции eval();
																код функции — код, выполняющийся в теле функции;
																глобальный код — код, не выполняющийся в рамках какой-либо функции.

Как изменить this внутри функции? (5 способов)      call, apply,  bind, new, вызвать на обьекте


чем различаются .call / .apply / .bind    call - вызов функции с пометкой контекста this
										  apply - вызов функции с переменным количеством аргументов и с подменной контекста
										  bind - создает обертку над функцией, которая подменяет контекст этой функции
										         (не вызывает функцию, а лиш возвращает обертку)

Что такое сигнатура функции?      - характеристическая часть определения функции в программировании.

Чем характеризуется функция?      - идентификатором, аргументы, переменные

Что такое прототип?      - свойсво наследуемое от Object, нужное для предоставления базового набора функциональных возможностей классу	

Как работает конструктор? Что происходит при вызове со словом new ?   - он рабоатет как функция, которая создает новый обьект. Функция вызваная через new становится конструктором

Как происходит чтение свойств из объекта?    -   obj.property  или obj.["property"]

Как происходит запись свойств в объект?    - через присвание нового значения свойству. obj.property = "string"

Как проверить на принадлежность классу?    -   с помощью оператора instanceof

Как работает instanceof ?     - rabbit instanceof Rabbit  -> возвращает true или false

4е принципа ООП       - абстракция, инкапсуляция, наследование, полиморфизм

Виды полиморфизма. И их объяснение   - параметрический - определяется функцию и тип данных обобщенно, так что значения обрабатываются идентично вне зависимости от их типа
									 - ad-hoc - перезагрука функций и методов, привеение типов
									 - полиморфизм подтипов - является обобщенной формализацией подтипизации и наследования
 
Событийный цикл в javascript   - стек, куча, очередь. Запуск до завершения, добавление событий в очередь, нулевые задержки, связь нескольких потоков между собой

Что такое фаза захвата / capturing ?     -    обработка событий должна спускаться по элементам («захватывать» их) вниз по DOM-дереву

Что такое фаза всплытия / bubbling ?    -   порядок выполнения событий должен подниматься («булькать») вверх по структуре DOM-дерева

Как подписаться на событие документа / html элемента?     - Подписаться через код HTML (не рекомендуется использовать).
													      -	Через код JavaScript с помощью задания элементу свойства on[событие], где [событие] - это имя определённого события.
														  - Через код JavaScript с помощью специального метода addEventListener.

Что такое Функция высшего порядка?    - функция, принимающая в качестве аргументов другие функции или возвращающая другую функцию в качестве результата. 

Что такое синхронный / асинхронный код?   -  синхронный код выполняется в том же потоке, в котором вызван метод. Асинхронный код выполняется в отдельном потоке

Что такое каррирование ?    -  преобразование функции от многих аргументов в набор функций, каждая из которых является функцией от одного аргумента.

Что такое паттерн цепочка ? Как реализовать?     -  паттерн поведения, выстраивающий объекты составных частей приложения связанными между собой по цепочке, для передачи запроса на обработку от более низких, детализированных слоев системы к более высоким глобальным.

В чем разница объявления методов в конструкторе и на .prototype ?    - через конструктор пожирается больше памяти, чем в прототипе

Что такое 'полифилл'?   -  это библиотека, которая добавляет в старые браузеры поддержку возможностей, которые в современных браузерах являются встроенными. (например библиотека ES5 shim)
